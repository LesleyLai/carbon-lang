`carbon-mode` provides Carbon syntax highlighting for Emacs.

## Installation

Currently only manual installation is supported.

### Manual installation

```elisp
(add-to-list 'load-path "/path/to/carbon-mode/")
(autoload carbon-mode "carbon-mode" nil t)
```
